# hotel-booking-demo
Hotel booking app demo with Spring boot Backend and Angular 10 FrontEnd App

Tech used:
- Angular 10
- Bootstrap
- Java Spring Boot
- Hibernate
- Maven

## Features
- User Authentication
- Multiple user profiles - Admin, Guest
- Admin
  - Manage all reservations
  - Manage rooms and prices
- User
  - Make reservations
  - Check own reservations
